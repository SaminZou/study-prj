# 下划线转驼峰

hello_world_samin -> helloWorldSamin

1. 用 Ctrl + D 选中所有需要修改的下划线

2. Shift + 方向键右，选中下划线后一个单词

3. Ctrl + Shift + P

4. 输入 "Uppercase"，回车

5. Backspase 删除前面的下划线
