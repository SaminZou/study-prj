# 模型分类

## 思维构建

> AI 要完成的事：模式识别（分类、检测、识别、判断）

- 生成型（造文字、造图、造语音）
- 推理型模型（逻辑、数学、规划、因果）
- 判断型（判断对错）

## 数据类型

| 类型     | 处理对象    |
| ------ | ------- |
| 文本模型   | 文本      |
| 视觉模型   | 图片 / 视频 |
| 语音模型   | 声音      |
| 时间序列模型 | 时序数据    |
| 多模态模型  | 多种数据    |


## 学习分类

| 类型   | 任务   |
| ---- | ---- |
| 分类   | 判断类别 |
| 回归   | 预测数值 |
| 聚类   | 自动分组 |
| 异常检测 | 发现异常 |
| 推荐   | 推荐内容 |
| 生成   | 生成数据 |


## 模型能力

| 类型    | 含义   |
| ----- | ---- |
| 专用模型  | 一个任务 |
| 多任务模型 | 一类任务 |
| 通用模型  | 很多任务 |

## 训练方式

| 类型    | 特点     |

## 规模

# 模型结算单位

| 模型类型                     | 标准结算单位          | 备注        |
| ---------------------------- | --------------------- | ----------- |
| XGBoost/LightGBM/ 结构化预测 | **调用次数 / 样本数** | 最常用      |
| 大语言模型（LLM）            | **Token**             | 输入 + 输出 |
| 语音转文字（ASR）            | **分钟 / 秒**         | 按音频时长  |
| 语音合成（TTS）              | **字数 / 秒**         | 国内常用字  |
| 图像生成 / 处理              | **张数 + 分辨率**     | 最直观      |
| 视频生成 / 处理              | **秒数 + 分辨率**     | 行业统一    |
