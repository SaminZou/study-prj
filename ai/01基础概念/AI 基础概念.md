# 模型训练框架

| 算法          | 使用框架                 | 适用场景   |
| ----------- | -------------------- | ------ |
| 随机森林        | scikit-learn         | 小数据预测  |
| XGBoost     | XGBoost              | 工业预测常用 |
| LSTM        | TensorFlow / PyTorch | 时间序列预测 |
| Prophet     | Prophet              | 周期趋势预测 |
| Transformer | PyTorch              | 大规模预测  |

工业能源公司通常使用 XGBoost 或 LSTM

# 模型分类

## 思维构建

> AI 要完成的事：模式识别（分类、检测、识别、判断）

- 生成型（造文字、造图、造语音）
- 推理型模型（逻辑、数学、规划、因果）
- 判断型（判断对错）

## 数据类型

| 类型     | 处理对象    |
| ------ | ------- |
| 文本模型   | 文本      |
| 视觉模型   | 图片 / 视频 |
| 语音模型   | 声音      |
| 时间序列模型 | 时序数据    |
| 多模态模型  | 多种数据    |


## 学习分类

| 类型   | 任务   |
| ---- | ---- |
| 分类   | 判断类别 |
| 回归   | 预测数值 |
| 聚类   | 自动分组 |
| 异常检测 | 发现异常 |
| 推荐   | 推荐内容 |
| 生成   | 生成数据 |


## 模型能力

| 类型    | 含义   |
| ----- | ---- |
| 专用模型  | 一个任务 |
| 多任务模型 | 一类任务 |
| 通用模型  | 很多任务 |

## 训练方式

| 类型    | 特点     |
| ----- | ------ |
| 监督学习  | 有标签    |
| 无监督学习 | 没标签    |
| 半监督学习 | 部分标签   |
| 自监督学习 | 自动生成标签 |
| 强化学习  | 奖励机制   |

## 规模

| 类型   | 特点    |
| ---- | ----- |
| 小模型  | 几 MB  |
| 中型模型 | 几百 MB |
| 大模型  | 数十 GB |
| 基础模型 | 超大规模  |
