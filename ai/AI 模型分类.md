# 思维构建

> AI 要完成的事：模式识别（分类、检测、识别、判断）

- 生成型（造文字、造图、造语音）
- 推理型模型（逻辑、数学、规划、因果）
- 判断型（判断对错）

# 数据类型

| 类型     | 处理对象    |
| ------ | ------- |
| 文本模型   | 文本      |
| 视觉模型   | 图片 / 视频 |
| 语音模型   | 声音      |
| 时间序列模型 | 时序数据    |
| 多模态模型  | 多种数据    |
