# 进阶文档

## PRD（Product Requirement Document）

- 详细功能说明（功能清单、优先级、功能目的、功能详细说明）

- 业务流程、用例

- 业务规则

- 原型图

- 数据要求（输入输出、极限范围、数据格式、数据闭环等）

## BRD（Business Requirement Document）

- 项目背景

- 市场分析

- 团队

- 产品路线

- 财务计划

- 竞争对手分析

## MRD（Market Requirement Document）

- 目标市场分析（目标、模型、特征、趋势）

- 目标用户分析（用户描述、用户使用场景、用户分类统计、核心用户、用户分类分析、竞争对手分析、产品需求概况）
