1. 整数拆分

> 题目解析：给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。

数学推导结果：
1. 当所有拆分出的数字相等时，乘积最大
1. 最优拆分数字为 3

```java
public int integerBreak(int n) {
    if(n <= 3) return n - 1;
    int a = n / 3, b = n % 3;
    if(b == 0) return (int)Math.pow(3, a);
    if(b == 1) return (int)Math.pow(3, a - 1) * 4;
    return (int)Math.pow(3, a) * 2;
}
```
