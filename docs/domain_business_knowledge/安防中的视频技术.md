# 基本概念

## 图像 Image

## 视频 video

- 时间上连续的图像组成视频：Image -> Video
- 视频中的某一幅图像成为一帧（Frame）
- 帧率（Frame Rate）FPS -> 每秒的帧数

## 码流 Bit Stream

- 将图像压缩形成的数据
- 码率（Bit Rate） bps/Bps -> 对码流进行量化
- 码率类型：定码率（CBR）、变码率（VBR）
- 码流类型：视频流、音频流、复合流
